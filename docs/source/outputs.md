# Outputs

Axion-HDL generates the following outputs:

## VHDL Register Module

`<module>_axion_reg.vhd` - A fully compliant AXI4-Lite slave module.

## C Header

`<module>_regs.h` - Contains address offset macros, field masks, and helper functions.

## Documentation

`register_map.md` - Markdown documentation of the register map.

## XML/YAML/JSON

Re-exportable register definitions for other tools.
